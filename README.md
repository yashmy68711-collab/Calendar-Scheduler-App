# Python Calendar Scheduler App

## About

A simple CLI Calendar Scheduler application built using Python.
This application helps users manage tasks by adding, viewing, and deleting scheduled tasks with dates.

---

## ✨ Features

✔️ Add tasks with dates
✔️ View scheduled tasks
✔️ Delete tasks
✔️ Simple task management
✔️ Input validation

---

## 🛠️ Tech Used

* Python
* Lists
* Dictionaries
* Object-Oriented Programming (OOP)

---

## 🚀 How to Run

1. Download the project
2. Open the Python file
3. Run the program
4. Choose options from the menu
5. Manage your scheduled tasks
